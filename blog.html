Question-2: What is the difference between innerHTML, innerText, and textContent?

The textContent property: This property is used to get or set the text content of the specified node and its descendants.
The innerHTML property: This property is used to get or set the HTML content of an element.
The innerText property: Unlike innerHTML, this property reads only the text that is visible on the screen, ignoring HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated. It allows us to attach a single event listener for elements that exist now or in the future.

Question-4: What is event bubbling in the DOM?

Bubbling in the DOM is an event propagation method. It starts with an event triggering on the innermost element and then proceeds to trigger the same event on its parent element and then its parent's parent element and so on.
